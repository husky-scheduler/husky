# Husky Docusaurus site

This directory contains the Docusaurus application for Husky documentation.

The generated site is intentionally separated from the Go code and dashboard code:

- site application source lives in this directory
- canonical product documentation source stays in `../docs`

## Local development

```bash
cd docs-site
npm install
npm start
```

## Production build

```bash
cd docs-site
npm install
npm run build
npm run serve
```

## Notes

- The docs plugin reads markdown directly from `../docs`.
- The homepage is implemented in `src/pages/index.jsx`.
- The sidebar is autogenerated from the repository-level docs tree.
